### 2. Add Carrying Cost Calculation

```python
def _calculate_carrying_cost(self, estimated_repairs: float, 
                            holding_months: int = 5) -> float:
    """Calculate carrying cost for hold period."""
    # Interest on acquisition + repairs during construction
    interest_rate_annual = 0.10  # Hard money @ 10%/year
    blended_amount = estimated_repairs * 0.5  # Cost phases in over time
    monthly_rate = interest_rate_annual / 12
    
    carrying_cost = blended_amount * monthly_rate * holding_months
    contingency = carrying_cost * self.CARRYING_COST_CONTINGENCY
    
    return carrying_cost + contingency
```

**Impact**: Subtracts $8-12K from profit, identifies marginal deals.
